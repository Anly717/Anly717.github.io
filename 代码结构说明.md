# 代码结构说明（大肥羊校园闲置物品交换平台 Demo）

本文档用于说明当前静态原型的代码拆分结构、各文件职责、核心数据/函数与运行方式。

---

## 1. 项目文件结构

> 说明：以下为与网页运行直接相关的核心文件。

- [index.html](index.html)：页面结构（HTML）
- [styles.css](styles.css)：页面样式（CSS）
- [app.js](app.js)：页面交互逻辑（JavaScript）

静态资源（被页面引用）：
- [logo.jpg](logo.jpg)：页头 Logo
- [background.jpg](background.jpg)：页面背景图
- [高数教材.avif](高数教材.avif)、[有线键盘.webp](有线键盘.webp)、[小台灯.jpg](小台灯.jpg)、[口红.jpg](口红.jpg)、[U盘.jpg](U盘.jpg)、[收纳箱.jpg](收纳箱.jpg)：物品图片

---

## 2. 运行方式（无需安装环境）

- 直接双击打开 [index.html](index.html) 即可运行。
- 需要保证 `index.html / styles.css / app.js` 和图片资源在同一目录下，且文件名不变（或同步修改引用路径）。

---

## 3. HTML 结构（index.html）

[页面入口](index.html) 的职责：
- 定义页面模块与 DOM 结构
- 通过外链方式加载样式与脚本：
  - `<link rel="stylesheet" href="./styles.css" />`
  - `<script src="./app.js" defer></script>`（`defer` 确保 DOM 解析完成后再执行 JS）

主要区域（按页面从上到下）：
- 顶部栏（`.topbar`）：Logo/标题、搜索框、重置按钮
- 首屏两列（`.hero`）：
  - 推荐轮播（按钮：`#btnPrev` / `#btnNext`）
  - 分类导航（按钮：`.chip[data-cat]`、发布入口 `#btnGoPublish`）
- 列表区（`.grid`）：
  - 最新发布列表容器：`#latestList`
  - 热门推荐列表容器：`#hotList`
- 详情区（`#detail`）：详情渲染容器 `#detailGrid`、联系模块 `#contact`
- 发布区（`#publish`）：表单 `#publishForm`（含填充示例按钮 `#btnFillDemo`）
- 浮动按钮（`.floating`）：`#btnFloatPublish` / `#btnFloatTop`

---

## 4. CSS 结构（styles.css）

[样式文件](styles.css) 的职责：
- 定义全局主题变量（`:root`）与基础排版
- 负责背景图层、布局网格、卡片样式与响应式

关键点：
- 主题变量：`:root` 内的 `--brand / --muted / --line ...`
- 背景图与透明度：使用 `body::before` 单独图层控制 `opacity: 0.3`，避免内容一起变透明
- 容器宽度/留白：`.container` 控制最大宽度与 padding
- 两列对齐：`.hero` 与 `.grid` 都采用 `grid-template-columns: 1fr 1fr`
- 移动端响应式：`@media (max-width: 860px)` 下两列改为一列

---

## 5. JavaScript 结构（app.js）

[脚本文件](app.js) 的职责：
- 维护页面状态（搜索词/分类/选中物品/轮播索引/物品列表）
- 将数据渲染为卡片列表与详情内容
- 绑定按钮、输入框、表单等交互事件

### 5.1 状态与数据

- `state`
  - `q`：搜索关键词
  - `cat`：当前分类（默认“全部”）
  - `selectedId`：当前选中的物品 id（用于详情与联系）
  - `carouselIndex`：轮播索引
  - `items[]`：物品数组（每个物品包含 `id/title/category/image/price/condition/location/time/seller/contact/desc`）

### 5.2 DOM 引用

- `el`：集中缓存常用 DOM 节点（例如 `#q #btnSearch #latestList #hotList #detailGrid ...`），避免重复查询。

### 5.3 核心函数

- 数据与过滤：
  - `itemMatches(item)`：判断物品是否符合搜索词与分类
  - `sortLatest(items)`：按 `time` 倒序（演示版：字符串比较）
  - `sortHot(items)`：演示热度排序（免费优先 + 时间 + 价格）

- 渲染：
  - `renderCard(item, { hot })`：生成单张卡片 HTML（含缩略图）
  - `renderLists()`：渲染“最新发布/热门推荐”两列列表
  - `openDetail(id)` / `closeDetail()`：渲染/收起详情

- 搜索与筛选：
  - `applySearch()`：读取搜索框并刷新列表
  - `setCategory(cat)`：设置分类并刷新列表
  - `resetAll()`：清空搜索/分类/详情并回到顶部

- 联系方式：
  - `showContact()`：展示联系模块
  - `copyContact()`：复制联系方式（部分浏览器可能因权限限制失败，会弹窗提示手动复制）

- 发布回流：
  - `addNewItemFromForm()`：校验表单并插入新物品到列表顶部

- 轮播：
  - `renderCarousel()` / `nextCarousel(delta)`：轮播内容渲染与切换

### 5.4 事件绑定与初始化

- 事件绑定集中在文件底部：搜索/重置/分类 chip/详情收起/联系/复制/发布/浮动按钮/轮播切换/表单提交等。
- 初始化：
  - `renderCarousel()`
  - `renderLists()`
  - `setInterval(() => nextCarousel(1), 6000)`：每 6 秒自动轮播

---

## 6. 常见修改入口（快速定位）

- 想改“物品内容/图片”：编辑 [app.js](app.js) 中 `state.items`（尤其是 `image` 路径）
- 想改“整体布局/留白/字体”：编辑 [styles.css](styles.css)（`.container`、`body`、各模块字号）
- 想改“模块结构/按钮文案”：编辑 [index.html](index.html)

---

## 7. 注意事项

- 图片文件名包含中文（如 `高数教材.avif`）：在本地直接打开通常没问题；如后续部署到服务器，建议统一为英文文件名并同步修改 `app.js` 中的 `image` 路径。
- 当前为静态演示版：不接入后端、不做支付/跨校/物流；发布数据仅保存在内存里，刷新页面会恢复初始模拟数据。
